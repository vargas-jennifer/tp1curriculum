# TP1: Curriculum Vitae

Alumna: **Jennifer Vargas**

Este TP consiste en un resumen de mi perfil como estudiante de Diseño y Comunicación Visual. El código lo realicé en Visual Studio Code en un tamaño de pantalla de 1920x1080px.

Para la construcción del sitio utilicé lo siguiente:
* Lenguajes: **HTML5 y CSS**
* Diseño visual: **Adobe Illustrator y Photoshop**

(Esto último lo utilicé para diseñar un prototipo del sitio y editar los tamaños y resoluciones de las imágenes.)

Para la organización de la estructura: 
* **'index.html'**: estructura principal
* **'estilos.css'**: hoja de estilos 
* **'/imágenes'**: carpeta de contenido gráfico (fotos, íconos, tramas).
